Для того, чтобы программа работала, нужно сделать следующие шаги:

1. Создать базу данных в PostgreSQL.

2. Прописать настройки базы в файл "application.properties" по пути src\main\resources

3. Запустить программу. Пи запуске создадуться необходимые таблицы.

4. Выполнить к базе данных следующий запрос:
INSERT INTO public.u_roles(id, name) VALUES (1, 'ROLE_USER'), (2, 'ROLE_ADMIN'); 

5. Перезапустить программу.

Для того, чтобы сделать пользователя администратором, нужно выполнить запрос:
UPDATE users_roles SET roles_id = 2 WHERE user_id = "id пользователя"


Для того, чтобы сделать администратора пользователем, нужно выполнить запрос:
UPDATE users_roles SET roles_id = 1 WHERE user_id = "id пользователя"
